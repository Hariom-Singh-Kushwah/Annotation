#!/home/ssapl/.virtualenvs/cds2/bin/python3

To automatically annotate a video with one object for PASCAL and YOLO model training.
1. Run the program -> The video starts playing.
2. Press 'g' to pause video -> Paused frame is dispayed.
3. Press left mouse button and drag to fit bounding box across the object.
4. If satisifed with bounding box Press 'c' or press 'r' to back to step 3.
5. Output will be generated in the given path as Image, xml, and txt files.
